# RedactUsers

Веб-приложение для управления пользователями с использованием официального драйвера MongoDB. Позволяет добавлять, искать, редактировать и удалять пользователей.

## Технологии

- **Back-end**:
  - Node.js
  - Express
  - MongoDB
  - xss, validator

- **Front-end**:
  - HTML/CSS/JavaScript
  - Fetch API
  - DOM манипуляции
  - PurifyJS

## Описание

**Front-end**:
- Взаимодействие с сервером осуществляется через AJAX запросы (fetch), что позволяет получать, редактировать данные без перезагрузки страницы.
- Проверка данных с помощью регулярных выражений для обеспечения корректности введенных данных.
- Защита отправляемых данных с помощью библиотеки PurifyJS для предотвращения XSS-атак.

**Back-end**:
- Сервер построен на Express.js.
- Структура проекта разделена на public, routes и controllers для лучшей организации кода.
- Использование библиотек xss и validator для проверки и очистки всех входящих данных, что повышает безопасность приложения.
- Взаимодействие с базой данных MongoDB осуществляется через официальный драйвер MongoDB.
